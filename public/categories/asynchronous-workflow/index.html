Motivation

We’ve finally reached this point. We can no longer keep adding more operations into our API endpoint workflow. It may be because we are reaching the API Gateway limit or because we are feeling ashamed of having a request that takes more than n seconds where n is an integer that represents a time in seconds that has been steadily increasing in the last months. Either way, it is time to build an asynchronous workflow wich includes a change in your way of thinking about workflows and all that added complexity.
